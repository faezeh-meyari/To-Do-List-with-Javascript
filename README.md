# To Do List Application

A dynamic and interactive To Do List application built with HTML, CSS, and Vanilla JavaScript.

This project shows practical front-end development concepts including DOM manipulation, state management, event handling, and persistent data storage using the browser's LocalStorage API.

The application allows users to efficiently manage daily tasks with filtering, editing, and completion tracking features.

## Project Overview
This project was created to improve my overall JavaScript skills and gain hands-on experience building interactive web applications.

Throughout the development process, I practiced:

- Working with DOM and updating content dynamically
- Handling user interactions and events
- Managing application behavior based on different conditions
- Conditional filtering logic
- Persistent state using LocalStorage
- Implementing basic task management features such as creating, updating, and removing items

All task data is stored locally in the browser, meaning tasks remain saved even after refreshing the page.

---

## Features

- Add new tasks
- Edit existing tasks
- Delete individual tasks
- Clear all tasks at once
- Mark tasks as completed
- Filter tasks by:
  - All
  - Pending
  - Completed
- Persistent data using LocalStorage
- Dynamic UI updates without page reload
